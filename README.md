# Short commands 
> NOTE: This package is still in production at 0.1.0 it will become downloadable at version 1.0.0. Contribution will be appreciated.

This is a CLI tool that is design to assign a fixed short equivalent of the frequently used commands you like. The commands are stored in a JSON object for now(_Will be looked into_). Look at [Invairiants](./INVARIANTS.md).

## Installation
> Required packages: Git and Cargo

1. Clone the repo
```bash
git clone https://github.com/emma31-dev/short_commands
```

2. Start the program
```bash
cargo run new
# OR
cargo run cc
# where cc is an example short command that's created
```
